# Task & Calendar Management System

이 프로젝트는 웹 브라우저의 로컬 스토리지를 활용하여 업무와 일정을 효율적으로 관리할 수 있는 **웹 기반 통합 태스크 관리 시스템**입니다. 별도의 서버 구축 없이 즉시 실행 가능하며, 직관적인 UI와 강력한 필터링 기능을 제공합니다.

## 🚀 주요 기능

* **직관적인 태스크 관리**: 업무 생성, 수정, 삭제(CRUD) 및 상태 관리
* **진행률 최적화**: 슬라이더와 수치 입력의 8:2 비율 인터페이스를 통한 정밀한 진행률 제어
* **분류 및 필터링**: 카테고리별 필터링 및 우선순위(High/Medium/Low) 지정
* **데이터 관리**: 브라우저 로컬 스토리지를 통한 데이터 자동 저장 및 CSV 파일 내보내기/가져오기
* **사용자 친화적 디자인**: 깔끔한 엔터프라이즈급 UI 적용

1. 다운로드된 폴더 안의 `task-manager.html` 파일을 웹 브라우저(Chrome, Edge 등)로 엽니다.
2. 별도의 서버 설정 없이 즉시 사용을 시작할 수 있습니다.

## ⚙️ 디자인 포인트

본 시스템의 입력 폼은 사용성을 극대화하기 위해 다음과 같은 CSS 규칙이 적용되어 있습니다.

* **Flexbox 레이아웃**: `.range-wrapper`를 통해 슬라이더와 숫자 입력창을 가로로 정렬합니다.
* **비율 최적화**:
* `input[type="range"]`: `flex: 8` (공간의 80%)
* `.num-progress-input`: `flex: 2` (공간의 20%)
* 이를 통해 화면 크기에 상관없이 항상 일정한 비율의 직관적인 입력을 지원합니다.
